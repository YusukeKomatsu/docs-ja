DCPキューの確認

DCPキューは、バケット間でデータ変更をストリームするために使用する、データベース変更プロトコル(DCP)の情報を提供します。

DCPキューの情報は各ノードのServer Nodesタブから参照できます。

  1. Server Nodes > node linkに移動します。
  2. (三角形を開くのではなく)ノードのリンクをクリックします。
  3. DCP QUEUESモジュールの三角形を開きます。
  4. DCP情報上にマウスを乗せると、バケット分析に関する説明が確認できます。
  5. バケット分析モジュールの三角形を開くと、サーバ単位の情報を確認できます。
