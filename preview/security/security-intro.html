セキュリティ

Couchbase Server 3.0は、アプリケーションや管理者向けに安全なネットワーク経由のデータアクセスをサポートしています。

Security is very important for big data applications, which process a large amount of unstructured data coming in big volumes and at high speed.

Couchbase Server provides security features associated with administrative access and client access via Secure Socket Layer (SSL).

  1. Encrypted administrative access
    Encrypted administrative access enables HTTPS access for Couchbase Web Console and REST API using the Secure Socket Layer (SSL) authentication.
  2. Encrypted data access
    Couchbase Server client libraries support client-side encryption using the Secure Sockets Layer (SSL) protocol.
  3. Best security practices
    Best practices for security consist of guidelines and tips to provide security features throughout the network.
