Diskinfo

The cbstats diskinfo command provides data and file size information.

CLI command and parameters

cbstats diskinfo

Syntax

Request syntax:

cbstats [localhost]:11210 diskinfo

Example

Example request:

cbstats 10.5.2.54:11210 diskinfo

Response

Example response:

 ep_db_data_size: 121104
 ep_db_file_size: 19036812
