3.0の新機能

Couchbase Server 3.0はスケーラビリティ、パフォーマンス、可用性、信頼性、管理の容易性、セキュリティに対する、非常に重要な機能拡張を含むメジャーリリースです。

Couchbase Server 3.0は、最も高性能でスケーラブルなNoSQLデータベースとして、 ミッションクリティカルなエンタープライズアプリケーションにおいて、Couchbaseがリードする領域を広げます。 エンタープライズ用途の優れた新機能を追加するだけではなく、3.0では、最適なNoSQLプロダクトとしてご利用いただくため、 開発者や管理者がCouchbase Serverをより簡単に扱うための新機能を追加しています。

新機能

次のリストはCouchbase Server 3.0の主要な機能拡張、新機能です:

  • 基盤のエンハンスメント:
    • データベース変更プロトコル(Database Change Protocol, DCP)
    • 共有スレッドプール

      DCPに関する統計情報について、WebコンソールはDCPキューの確認を、CLIはDCPをご参照ください。

  • バケットとディスクI/Oに関するエンハンスメント:
    • バケットメタデータの除去

      バケットメタデータの除去について、Webコンソールはメモリ内メタデータの管理を、 CLIはメタデータ除去ポリシーの設定を、 REST APIはメタデータ除去設定をご参照ください。

    • ディスクI/O優先度

      ディスクI/O優先度設定の変更方法は、Webコンソール経由の場合Managing disk I/O priorityを,CLIからはSetting bucket priorityを、REST APIからはSetting disk I/O priorityをご参照ください。

  • サーバメンテナンスの向上:
    • グレイスフルフェイルオーバ

      グレイスフルフェイルオーバについて、WebコンソールはFailing over a node、CLIはFailing over nodes、REST APIはSetting graceful failoverをご参照ください。

    • デルタノードリカバリ

      デルタノードリカバリについて、WebコンソールはRecovering a node、CLIはRecovering nodes、REST APIはSetting recovery typeをご参照ください。

    • 差分バックアップとリストア

      差分バックアップとリストアについて、CLIはBacking up incrementally、Restoring incrementallyをご参照ください。

  • Stream-based viewsによるViewの向上。
  • Stream-based XDCRとXDCR pause and resume replicationによるXDCRの向上:

    XDCRレプリケーションの一時停止と再開について、 WebコンソールはPausing XDCR replication、CLIはPausing XDCR replication streams、REST APIはPausing XDCR replication streamsをご参照ください。

  • Encrypted administrative accessとEncrypted data accessによるセキュティ向上。
  • Cluster-wide diagnosticsによるログとログ収集の向上:

    診断情報の収集について、WebコンソールはManaging diagnostics、CLIはDiagnostics with couchbase-cliとcbcollect_info toolをご参照ください。
