Dispatcher

The cbstats dispatcher command gets statistics from reader, writer, auxio, and nonio worker threads.

Description

This provides the stats from AUX dispatcher and non-IO dispatcher, and from all the reader and writer threads running for the specific bucket. Along with stats, the job logs for each of the dispatchers and worker threads is also made available.

The following stats are available for the workers and dispatchers:

Stat Description
cur_time The current timestamp.
runtime The amount of time it tool for the job to run in milliseconds.
state Threadsâ€™s current status: running, sleeping, etc.
task The activity the thread ran during that time.
waketime The time stamp when the job started.

Syntax

Request syntax:

cbstats [hostname]:11210 dispatcher

Example

Example request:

cbstats 10.5.2.54:11210 dispatcher
